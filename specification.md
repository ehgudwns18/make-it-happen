# 서비스 기획서: Make It Happen (소셜 미디어 앱)

## 1. 서비스 목적
**Make It Happen**은 시각적 스토리텔링을 통해 사용자 간의 소통을 장려하는 현대적인 모바일 중심의 소셜 미디어 애플리케이션입니다. 사용자는 자신의 일상을 공유하고, 좋아요와 댓글로 커뮤니티와 소통하며, 트렌디한 시각적 피드를 탐색할 수 있습니다.

## 2. 핵심 기능

### 📸 피드 및 인터랙션 (Feed & Interaction)
*   **비주얼 중심 피드**: 여러 장의 이미지를 슬라이드 형식으로 지원하는 넓은 화면의 이미지 피드.
*   **직관적인 인터랙션**:
    *   **더블 탭 좋아요**: 화면을 두 번 터치하여 빠르게 좋아요를 표시하는 제스처 및 하트 애니메이션.
    *   **좋아요, 댓글, 저장**: 게시물 하단에서 바로 접근 가능한 표준 소셜 액션 버튼.
    *   **타임스탬프**: 상대적인 시간 표시 (예: "2시간 전").

### 💬 소통 및 참여 (Comments & Replies)
*   **슬라이드 업 댓글 창**: 피드 화면을 벗어나지 않고 자연스럽게 열리는 현대적인 드로어(Drawer) UI.
*   **대댓글 (답글)**: 댓글에 대한 답글을 들여쓰기로 시각적으로 구분하여 대화 흐름 지원.
*   **실시간 반응성**:
    *   댓글 작성 시 즉각적인 리스트 반영.
    *   "~님에게 답글 남기는 중" 표시 및 취소 기능.
    *   개별 댓글에 대한 좋아요 기능.

### ➕ 콘텐츠 제작 (Content Creation)
*   **매끄러운 작성 흐름**: 2단계로 구성된 콘텐츠 공유 프로세스.
    1.  **이미지 선택**: 최근 사진 또는 사용자 직접 업로드 이미지 선택 (사용자 업로드 지원).
    2.  **게시물 에디터**: 문구 작성, 해시태그 추가.
*   **글로벌 상태 관리**: 작성된 새 게시물은 별도의 새로고침 없이 메인 피드 상단에 즉시 반영.

## 3. 디자인 시스템 및 UX

### 시각적 정체성 (Visual Identity)
*   **스타일**: 콘텐츠에 집중할 수 있는 깔끔하고 미니멀한 스타일 (인스타그램 영감).
*   **타이포그래피**: 모바일 환경에서 가독성이 뛰어난 산세리프(San-serif) 폰트 사용.
*   **아이콘**: 선명하고 현대적인 느낌의 고품질 SVG 아이콘 (Lucide React) 활용.

### 사용자 경험 (UX)
*   **모바일 퍼스트**: 터치 인터랙션에 최적화된 레이아웃 및 버튼 크기.
*   **마이크로 인터랙션**: 앱에 생동감을 더하는 미세한 애니메이션 (페이드 인, 하트 팝업, 버튼 호버 효과).
*   **피드백**: 사용자의 행동에 대한 명확한 시각적 피드백 (좋아요 시 빨간 하트, 저장 시 채워진 아이콘 등).

## 4. 기술 아키텍처

### 기술 스택 (Technology Stack)
*   **프레임워크**: React 18 + Vite (빠른 개발 및 HMR 지원).
*   **언어**: TypeScript (타입 안정성 및 유지보수성 확보).
*   **스타일링**: Tailwind CSS (유틸리티 우선 스타일링) + shadcn/ui (접근성이 고려된 컴포넌트 프리미티브).
*   **런타임**: Bun.

### 코드 구조 개선 (Refactoring)
*   **Context API**: `PostContext`를 통해 전역 게시물 상태를 관리하여 동적인 콘텐츠 업데이트 구현.
*   **커스텀 훅 (Custom Hooks)**: `usePostInteraction`을 만들어 복잡한 로직(좋아요, 더블 탭 등)을 캡슐화하고 UI 컴포넌트를 순수하게 유지.
*   **컴포넌트 모듈화**:
    *   `CommentItem`: 개별 댓글 렌더링 로직 분리.
    *   `ImageSelector` & `PostEditor`: 복잡한 생성 페이지 로직을 관리하기 쉬운 하위 컴포넌트로 분리.

## 5. 향후 로드맵
*   **스토리 뷰어**: 휘발성 콘텐츠를 위한 전체 화면 몰입형 뷰어 구현.
*   **검색 및 탐색**: 태그 및 사용자 검색 기능 추가.
*   **백엔드 통합**: 영구적인 데이터 저장을 위한 데이터베이스 및 인증 시스템 연동.
